使用pipenv 安装第三包失败

环境

  • os: mac os
  • python: 3.7
  • pip: 18.1

错误提示

原因
使用的pip版本过高，可能为开发版本

解决方案
降级pip至18.0版本

  • https://github.com/pypa/pipenv/issues/2871
